# VCT Platform — Pitch Deck HTML

## File: `docs/vct-platform-pitch.html` (48KB)

Professional pitch-ready HTML document with dark/light theming, 7-tab navigation, operation diagrams, and subsystem interaction maps.

## Features

| Feature | Details |
|---------|---------|
| **Light/Dark Theme** | Toggle button (top-right), persists to localStorage |
| **7 Tab Navigation** | Tổng quan, Phân hệ, Quy trình, Tương tác, Vai trò, Tổ chức, Công nghệ |
| **6 Flow Diagrams** | Tournament lifecycle, Registration, Approval, Scoring, Club creation, Discipline |
| **Interaction Maps** | 9 data flows + 2 operational scenarios |
| **Ecosystem Diagram** | Visual 3×3 grid showing all 9 subsystems |
| **Org Hierarchy** | LĐ QG → LĐ Tỉnh → CLB → VĐV structure with user counts |
| **RBAC Matrix** | 12 roles × 10 resources with permission codes |
| **Counter Animation** | Stats animate on scroll |
