# 📜 Quy chế cấp Quốc gia

Thư mục này chứa các văn bản quy chế, quy định hiện hành cấp **quốc gia** liên quan đến hoạt động Vovinam - Việt Võ Đạo.

## Cấu trúc thư mục

```
national/
├── README.md              # File này
├── *.pdf                  # Các file PDF quy chế gốc
└── summaries/             # (Tùy chọn) Tóm tắt nội dung quy chế
```

## Hướng dẫn đặt tên file

Để dễ quản lý, hãy đặt tên file theo quy ước:

```
[NĂM]-[MÃ_VĂN_BẢN]-[TÊN_NGẮN].pdf
```

**Ví dụ:**
- `2024-QC01-quy-che-thi-dau-vovinam.pdf`
- `2025-QD15-quy-dinh-dan-cap-quoc-gia.pdf`
- `2024-HD03-huong-dan-to-chuc-giai.pdf`

## Lưu ý

- Các file PDF là văn bản gốc, **KHÔNG** được chỉnh sửa.
- Nếu cần cập nhật, hãy thay thế file và ghi chú trong bảng danh sách.
- Thư mục `summaries/` dùng để lưu bản tóm tắt dạng Markdown nếu cần tra cứu nhanh.
